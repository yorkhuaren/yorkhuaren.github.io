2016最新约克大学教室代码详解

作者：约克华人网 2016-07-10
  • 留学
  • 约克大学

新来的小伙伴们在看课表时可能会看到类似ATB, CP, EN/B之类的代码，很多小伙伴瞬时就惊呆了，这都是神马鬼？？？其实，由于约克大学的建筑比较多，所以大学给所有的建筑都编了一个简写代码，方便使用而已。 一定要收藏下喔，省得找不到教室上课迟到喔。。。

下面是约克大学官网给出的各个Building/教室的代码，官网链接在这里：https://www.york.ac.uk/admin/estates/buildcodes.htm

The University Central Space Record (Planon) holds a unique identifier for all internal and many external spaces. The structure of the code is explained in the following example.

The first letter(s) represent the building code - ‘B’ separated by use of a forward slash from the second letter which is the block code - ‘K’ thus:

B/K

The block code ‘C’ is separated from the space identifier numbers again by use of a forward slash thus:

B/K/

The space code leading number indicates the floor level:

B/K/0 (0 = ground floor)

This is followed by two final numbers (01 to 99) identifying the space which may, on occasions, be appended with a letter indicating a sub space or division:

B/K/001 or B/K/001A

therefore:

B/K/001 (space 01, on the Ground Floor of Biology Laboratories, Block K)

B/K/299 (space 99, on the Second Floor of Biology Laboratories, Block K)
